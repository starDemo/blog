## Shell中的文字处理
- JSON
```bash
# jq是一个Shell中的常用json对象化工具
# 安装
yum install -y jq
# 使用Demo Data见后文 如下命令获取
kubectl get secrets -n test test -ojson
########Data##########################
kubectl get secrets -n test test -ojson|jq '.data.passwd'
"aGVsbG8="
```

- String 字符串处理去除引号
```bash
kubectl get secrets -n test test -ojson|jq '.data.passwd'
"aGVsbG8="
kubectl get secrets -n cms3 test -ojson|jq '.data.passwd'|sed 's/\"//g'
aGVsbG8=
```
